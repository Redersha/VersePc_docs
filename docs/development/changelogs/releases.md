## v1.0.0 (2026年6月20日)

亲爱的小伙伴们，VersePC 启动器正式开放预览版啦！🎉

预览版可能存在较多 Bug，如果你比较在意稳定性，建议等待八月份发布的正式稳定版。

 功能介绍

版本管理 — 一键下载和管理多个 Minecraft 版本，支持原版、Forge、Fabric、NeoForge 等主流加载器
资源整合 — 内置模组、整合包、资源包、数据包、光影下载
联机功能 — 支持局域网联机和端口映射
账户管理 — 支持微软正版登录和离线模式
Java 管理 — 自动检测和管理 Java 环境
工具箱 — 皮肤查看、游戏修复等实用工具
控制台 — 实时查看游戏日志
实验性功能

实验性功能（AI 助手等）目前需要赞助后获取测试版才能解锁使用。 欢迎前往 爱发电 支持我们的发展

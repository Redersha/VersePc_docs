# 基本功能

主页是启动器的起始页面。由左侧侧边栏，顶部栏和游戏启动栏构成。

![主页 截图](https://r2.image-upload.app/ptImg/0bJjmA09.png)

## 左侧侧边栏

此区域包含了启动器的所有功能的入口。
在点击 `资源` 或 `设置` 时，按钮会展开以展示更多页面，就像下图。

![侧边栏 截图](https://r2.image-upload.app/ptImg/1lB2w941.png)

## 实例设置（局部生效）

在 `主页` 的 `已安装版本` 区域，会列出已有的游戏实例。

点击一个实例即可进入实例的设置页面。

可以在此设置实例的信息、启动选项、内存分配、游戏导出等。（此处的设置只影响单个实例，若要全局影响，见[设置页面]）

![设置页面 截图](https://r2.image-upload.app/ptImg/tI1nPzuJ.png)

## 下载 / 版本管理

你可以在此页面下载新的实例，或者管理已有的实例。

![下载页面 截图](https://r2.image-upload.app/ptImg/wUkDQaDO.png)

## 资源下载与管理

你可以在此页面下载 / 管理游戏资源。包括：模组、整合包、数据包、材质包、光影包。

搜索栏下方的筛选器可以帮助你找到所需的资源。

此页面的游戏资源来自 modrinth / Curseforge。

:::tip 注意侧边栏，在侧边栏切换资源类型
:::

![模组页面 截图](https://r2.image-upload.app/ptImg/pmLCHk6E.png)

### 模组收藏功能

在你逛模组的时候，可以顺手收藏一些 **感兴趣但现在不能安装** 的模组。

在模组详情页点击带有心型图标的按钮即可收藏。在 资源>模组 页面可以管理你的收藏。

## 联机功能

出于篇幅考虑，这部分被移到[多人游戏](/multiplayer/terracotta)

## 账户管理

启动器支持微软登录、外置登录（LittleSkin Yggdrasil/MCBBS）、离线账户。

在此界面可以管理你的账户。

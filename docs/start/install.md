# 1. 下载与安装

0. VersePc 启动器 的系统要求

| 平台 | 最低要求 | 推荐配置 |
|------|---------|---------|
| Windows | Windows 10 (x64) | Windows 11 |
| macOS | macOS 12 (Intel/Apple Silicon) | macOS 14+ |
| Linux | 64-bit 发行版 | Ubuntu 22.04+ / Arch |

1. 下载安装包

    1. 打开[下载链接](https://pan.quark.cn/s/02655f4bf804#/list/share)，进入文件夹。
    2. 选择**适合你设备的 系统 / 架构 的安装包**并下载。
        :::tip Mac 用户：
        * 如果设备为英特尔芯片的 Mac，选择文件名结尾为 `x64.dmg` 的。
        * 如果设备为 Apple Silicon 的 Mac，选择文件名结尾为 `arm64.dmg` 的。
        :::

2. 安装

    1. 运行你刚刚下载的安装包。
        :::tip Mac 用户：
        * 双击打开安装包后，在弹出的窗口中将 VersePc 拖拽到右侧`应用`文件夹（或`Applications`文件夹）中。
        :::
    2. 根据安装程序的指示完成安装步骤。
        :::warning 如果安装过程中出现 “无法安装” 或类似字眼
        你的设备可能在此之前就安装过了 VersePc。先删除旧版本遗留，再继续下载。
        :::

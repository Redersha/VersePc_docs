# 3.1 如何使用陶瓦联机
VersePc 内置了 [Terracotta | 陶瓦联机](https://github.com/burningtnt/Terracotta)（本项目与陶瓦联机制作组无关）。

![内置陶瓦联机页面 截图](https://r2.image-upload.app/ptImg/skXQVlOu.png)

1. 房主启动一个单人游戏，并开放至本地局域网。
2. 记下开放后游戏左下角给出的端口号。
3. 回到启动器，在`创建房间`填写端口号，点击`开始创建房间`。
4. 稍等一会，启动器将给出房间码，房主将其转发给朋友。

![](https://r2.image-upload.app/ptImg/7ALWQvqd.png)

5. 朋友使用和房主同版本的游戏实例（如果安装了非客户端模组，比如机械动力等，则所有人都需要安装这些模组）、并启动游戏。

6. 朋友在`加入房间`页面输入房间码，点击`加入房间`。

7. 朋友在游戏主菜单点击`多人游戏`，进入本地局域网世界即可游玩。
